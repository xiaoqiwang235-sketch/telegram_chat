## 🚀 快速启动指南

### 1. 环境准备
```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件填入 API Keys

# 初始化数据库
python scripts/migrations/001_create_initial_tables.py
```

### 2. 启动机器人
```bash
python -m src.main
```

### 3. 使用命令
- 私聊中直接发送消息即可
- 群聊中使用 @botname 获得回复
- 输入 `/help` 查看所有命令

## 🎓 技术决策总结

### 为什么选择这个技术栈？
1. **python-telegram-bot**: 最成熟的 Telegram Bot 框架
2. **MySQL + Faiss**: 混合存储策略，兼顾性能和成本
3. **pymysql**: 直接使用，符合用户要求
4. **asyncio**: 高并发异步处理
5. **OpenAI 嵌入**: 业界最好的嵌入模型

### 架构优势
- **分层清晰**: Handlers → Services → Memory → Database
- **易于测试**: 依赖注入，Mock 友好
- **可扩展性**: 模块化设计，易于添加新功能
- **生产就绪**: 完整的错误处理和日志
